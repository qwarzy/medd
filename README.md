# 🦷 Дневник здоровья зубов

Медицинское мобильное приложение для Android. Помогает следить за гигиеной полости рта и планировать визиты к стоматологу.

## 📱 Функции приложения

-  **Ежедневные напоминания** — push-уведомления в 9:00 и 21:00 о чистке зубов
-  **Отметка чистки** — кнопка «Я почистил зубы» с защитой от повторных нажатий
-  **Статистика** — отображение количества утренних и вечерних чисток за сегодня
-  **Запись к врачу** — добавление визита с выбором даты и времени
-  **Список визитов** — отображение всех добавленных визитов
-  **Удаление визитов** — возможность удалить ненужный визит

## 🛠 Технологии

- Android SDK (API 24+)
- Java
- Room Database
- WorkManager
- RecyclerView
- NotificationManager

## 🚀 Установка и запуск

1. Клонировать репозиторий: `git clone https://github.com/qwarzy/medd.git`
2. Открыть проект в Android Studio
3. Подключить телефон или запустить эмулятор
4. Нажать Run ▶
